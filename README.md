# TODO TASKS

### Projeto Criado em NodeJS (Typescript)

* Principais Libs:
  * express
  * inversify
  * jsonwebtoken
  * mysql
  * typeorm

### Execução

* Clonar o projeto:  
  ```git clone git@github.com:LRissi/todo_tasks.git```
* Executar o docker:  
  ```docker-compose -f "docker-compose.yml" up -d --build```
